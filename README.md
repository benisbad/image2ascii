# image2ascii
Converts an image's black-and-white equivalent into ASCII.

# Examples
![Alt](/smile.png "Smile.png")

will be converted to
```
   ($$B
   $$`        #$
   $          $$$#
      $#         $
      u$     #$(
             $ Y
         ]
         BB
       #( $
        $$$
  WB
  B$            $
   $           W$
   `$/        $$
    $$$#   ]$$$`
       C$$$$
```
(scaled down for readability)
